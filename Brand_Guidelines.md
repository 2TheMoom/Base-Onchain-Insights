# 🎨 Brand Guidelines: Social Engagement Analyzer

This document defines the visual identity for the "Based Score" dashboard to ensure a consistent user experience.

## 1. Color Palette
* **Primary (Base Blue):** `#0052FF` - The core color of the Base network.
* **Secondary (Success Green):** `#00D395` - Used for "High Based Scores" and active streaks.
* **Accent (Flame Orange):** `#FF4D00` - Specifically for the "Infinity Flame" and high-activity alerts.
* **Background:** Dark Mode (`#0F172A`) for a high-tech, developer-centric feel.

## 2. Typography
* **Primary Font:** Inter or Geist (Clean, modern sans-serif).
* **Data Font:** JetBrains Mono (For displaying wallet addresses and SQL-driven metrics).

## 3. Visual Elements
* **The "Based" Progress Bar:** A gradient ring transitioning from Blue to Green as the score increases.
* **Iconography:** Minimalist line icons (using Lucide-React) to keep the UI lightweight.

## 4. Tone of Voice
* **Professional yet Encouraging:** The app should feel like a high-end financial tool but celebrate user achievements with "Based" terminology.
