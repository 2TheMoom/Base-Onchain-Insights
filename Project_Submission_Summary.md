# 🏆 Project Submission Summary: Social Engagement Analyzer

**Project Name:** Social Engagement Analyzer (Based Score Dashboard)  
**Chain:** Base Mainnet  
**Sector:** Social Fi / On-Chain Reputation  

### 1. The Problem
On-chain activity is fragmented. It is difficult for protocols to distinguish between "Sybil/Bot" accounts and "Consistent Human Builders" who have been active for 200+ days.

### 2. The Solution
A "Read-Only" reputation dashboard that uses **Dune Analytics SQL** to calculate a **Based Score (0-100)**. We prioritize "The Infinity Flame"—a metric recognizing users with 240+ days of consecutive on-chain presence.

### 3. Key Features
* **Rabby-First Security:** Native simulation-friendly connectivity.
* **Protocol Diversity Engine:** Rewards users for exploring the full Base ecosystem.
* **Visual Identity:** Custom Kitsune-masked branding for a "Based" aesthetic.
* **Open Source:** Transparent SQL logic and React frontend structure.

### 4. Technical Stack
* **Frontend:** Next.js, Tailwind CSS, ConnectKit, Wagmi.
* **Data Layer:** Dune Analytics API (SQL/Trino).
* **Network:** Base (Layer 2).

### 5. Future Impact
This tool scales into a "Proof of Contribution" API, allowing other Base dApps to reward loyal users with early access or gas rebates based on their verified on-chain history.
