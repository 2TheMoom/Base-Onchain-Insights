# 📖 Base Ecosystem Glossary

A quick reference guide for builders and researchers analyzing the Base network.

### Technical Infrastructure
* **Unified Base Stack (`base/base`):** The self-managed, consolidated software stack Base is transitioning to in 2026.
* **Reth:** A high-performance Ethereum execution client written in Rust, designed for high throughput (1 Gigagas/s).
* **L2 (Layer 2):** A scaling solution (like Base) that inherits the security of Ethereum (Layer 1) while providing faster and cheaper transactions.
* **Sequencer:** The component that orders and batches transactions before submitting them to the Ethereum mainnet.

### On-Chain Metrics
* **Gas Price:** The cost of executing a transaction, measured in Gwei. Base gas fees are significantly lower than L1.
* **DEX (Decentralized Exchange):** Platforms like Aerodrome or Uniswap where users swap assets on-chain.
* **Smart Contract:** Self-executing code that powers decentralized applications (dApps).

### Social & Identity
* **Farcaster:** A decentralized social protocol heavily used by the Base community.
* **On-chain Reputation:** A user's standing based on their transaction history, NFT holdings, and protocol interactions.
* **Based:** A term used to describe individuals or projects that align with the values of the Base ecosystem.
