# 🤝 Contributing to Social Engagement Analyzer

First off, thank you for considering contributing! It’s builders like you who make the Base ecosystem thrive.

### 1. How Can I Contribute?
* **SQL Optimization:** Improve our Dune Analytics queries for faster or more accurate data retrieval.
* **UI/UX Improvements:** Suggest or implement better ways to visualize the "Based Score."
* **Bug Reports:** Open an issue if you find a wallet connection error or a data discrepancy.

### 2. Style Guidelines
* **Code:** Use clean, commented React/Next.js code.
* **Documentation:** Keep the "Based" tone—professional, clear, and community-focused.
* **SQL:** Ensure queries are optimized for the Base Mainnet and follow Dune's best practices.

### 3. Pull Request Process
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes with a clear message.
4. Push to the branch and open a Pull Request.

### 4. Community Standards
We follow a "Build in Public" philosophy. Be respectful, be helpful, and let's build the best reputation tool on Base together.
