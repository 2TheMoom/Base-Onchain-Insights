# 🎨 Visual Assets Manifest: Social Engagement Analyzer

This document serves as the "Source of Truth" for the visual identity and branding of the Based Score dashboard.

### 1. The Mascot (The Based Kitsune)
* **Description:** A purple-toned, Kitsune-masked character representing "On-Chain Stealth & Wisdom."
* **Usage:** Primary profile avatar and "Loading" animation state.
* **Format:** Transparent PNG / SVG (Found in `/public/assets/brand/`).

### 2. Color Palette (The "Based" Spectrum)
* **Primary Purple:** `#6D28D9` (Deep Violet) - Used for buttons and highlights.
* **Base Blue:** `#0052FF` (Official Base Color) - Used for connectivity indicators.
* **Infinity Orange:** `#F97316` (Flame Glow) - Used exclusively for 240+ day streak indicators.

### 3. Typography (Developer Aesthetic)
* **Primary Font:** **JetBrains Mono** (Monospace) - Used for all data points and wallet addresses.
* **Secondary Font:** **Inter** (Sans-serif) - Used for body text and instructions.

### 4. Component Visuals
* **Based Score Circle:** A 360-degree gradient ring that fills based on the 0-100 score.
* **Streak Flame:** An animated SVG that pulses when a user has a "Hot Streak" (7+ days).

### 5. Social Assets
* **Warpcast Header:** 1500x500px banner featuring the Kitsune mask and the "Check Your Basedness" tagline.
