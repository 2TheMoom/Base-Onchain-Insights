# 🔥 Streak & Achievement System

To encourage long-term "Based" behavior, the Social Engagement Analyzer implements a tiered badge system based on on-chain consistency.

### 1. The "Flame" Tiers (Daily Streaks)
* **Genesis Spark (7 Days):** The entry-level badge for new builders.
* **Based Habit (30 Days):** Awarded for a full month of active participation.
* **Centurion (100 Days):** Recognizes significant dedication to the ecosystem.
* **The Infinity Flame (240+ Days):** The elite tier for users with near-perfect consistency (inspired by Base Explorer Analytics).

### 2. Activity Milestones
* **Protocol Pioneer:** Interacted with 10+ unique smart contracts on Base.
* **Early Adopter:** Wallet created within the first 6 months of Base Mainnet.
* **Gas Contributor:** Contributed over 0.05 ETH in cumulative network fees.

### 3. Display Logic
These badges will be rendered as SVG icons on the user's "Based Profile Card." Each badge will include a metadata link to the transaction history that proves the achievement (Proof of Contribution).
